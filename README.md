# 🌍 airIQ – Air Quality Prediction System

## 🚀 Overview

airIQ is a machine learning-based web application that predicts the Air Quality Index (AQI) using environmental data such as pollutants and atmospheric conditions. The project integrates data analysis, model training, and a Flask-based web interface for real-time predictions.

---

## 🧠 Features

* 📊 AQI prediction using trained ML model
* 🌐 Interactive web interface (Flask)
* 📁 Clean project structure (models, scripts, notebooks)
* 📈 Data analysis using Jupyter Notebook
* ⚡ Fast predictions using serialized model (.pkl)

---

## 🛠️ Tech Stack

* Python
* Flask
* Scikit-learn
* Pandas, NumPy
* HTML, CSS
